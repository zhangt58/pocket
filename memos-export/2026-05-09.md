### 2023-12-31 [92]

## Imgui for Golang

- https://github.com/AllenDang/cimgui-go active go binding for cimgui
  - archived one: https://github.com/inkyblackness/imgui-go, while this fork has a few more commits: https://github.com/AllenDang/imgui-go.git
- https://github.com/AllenDang/giu

### UI Rendering
- `giu` adopts lazy rendering policy regarding UI updating, that is if no user events incomming, the UI pauses the refreshing after 15 frames [(1)](https://github.com/AllenDang/cimgui-go/issues/234) [(2)](https://github.com/AllenDang/cimgui-go/blob/2be2c77051cfb58b153ace3305c0014bf34c8552/glfw_backend.cpp#L23), the recommended way to force the refresh is calling `giu.Update()`[(3)](https://github.com/AllenDang/giu/issues/701), this is the so-called power-saving mode.
- While for the case of building real time data visualization application, power-saving mode is not a good fit, there should be a way to override it, since calling `giu.Update()` seems of high overhead cost.
