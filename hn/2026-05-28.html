#10

Rapira (Рапира) – Soviet programming language interpreter

⭐ 19 💬 9 👤 begoon

Rapira is a TypeScript/Bun interpreter for the Soviet educational programming language designed in the early 1980s by G. A. Zvenigorodsky as part of the Shkolnitsa school computing system for the Agat microcomputer. Faithful to the 1985 Agat dialect with Russian-only keywords, it features a lexer, parser, and tree-walking evaluator supporting tuples, sets, records, and the three-arrow procedure parameter scheme. Run it in-browser or via npx rapira.

🔗 https://github.com/begoon/rapira
