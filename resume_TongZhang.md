## Professional Experience

### [Facility for Rare Isotope Beams (FRIB), Michigan State University](https://frib.msu.edu) — East Lansing, MI, USA

**Staff Scientist** — *Oct 2023 – Present*  
**Scientist** — *Sep 2016 – Sep 2023*

- **Physics Application Development:** Lead design and development of high-level physics applications for accelerator commissioning, beam tuning, and operations — used daily by FRIB operators and physicists.
- **Control Systems & GUI Applications:** Built desktop and web-based applications for EPICS control systems, including PhantasyCalc (spreadsheet-integrated EPICS data tool) and OPI-Generator (automated control screen generation for CS-Studio/Phoebus).
- **Software Architecture:** Designed and deployed `phyapps-cloud` — a cloud platform for physics high-level application development, enabling distributed scientific computing workflows.
- **Data Management Systems:** Developed systems for large-scale data acquisition, storage, and retrieval including `PyArchAppl` (Python interface to EPICS Archiver Appliance).
- **ML/AI Computing Platform:** Maintained CUDA-based machine learning computing infrastructure for accelerator applications.
- **Legacy Code Maintenance:** Lead developer and maintainer of `LISE++` — a widely-used C++/Qt nuclear physics simulation code for rare isotope production.
- **Open Source Contributions:** Authored/maintained 10+ open-source packages including optimization frameworks and simulation codes.
